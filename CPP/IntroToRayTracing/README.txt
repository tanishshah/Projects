Some sample code for learning about ray-tracing
There are many bugs present in this code and features that
don't yet work as intended, however it served as a good chance to learn
about the basics of raytracing.

Based on the seminal work by Peter Shirley on Raytracing

https://github.com/RayTracing/raytracing.github.io
“Ray Tracing in One Weekend.” raytracing.github.io/books/RayTracingInOneWeekend.html
Author: Peter Shirley
Editors: Steve Hollasch, Trevor David Black
